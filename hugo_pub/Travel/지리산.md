### 차량 1대 최적 동선

> 비바체는 삼화실 쪽에 가까우므로, **차를 원춘부(2일차 종료지점)에 미리 주차**하는 게 가장 효율적

| 단계 | 내용 |
|------|------|
| 출발 당일 | 원춘부에 차 주차 → 택시로 비바체 이동 & 체크인 |
| 1일차 아침 | 비바체 → 택시 → 삼화실 (출발) |
| 1일차 낮 | 삼화실 → 대축 걷기 (16.7km) |
| 1일차 저녁 | 대축 → 택시 → 비바체 (숙박) |
| 2일차 아침 | 비바체 → 택시 → 대축 (출발) |
| 2일차 낮 | 대축 → 원춘부 걷기 (8.6km) → **차가 여기에 있음** |
| 귀가 | 원춘부에서 바로 귀가 |

- 택시 총 3회 (원춘부↔비바체 1, 비바체↔삼화실 1, 대축↔비바체 1)
- 원춘부~삼화실 사이 택시 이동 사전 콜밴 예약 권장 (산간 지역)

### 준비물

- 출발시
	- 신발 여벌, 핸드폰 충전기
- 산행시 
	- 양발, 모자 , 손수건, 물, 보조배터리 ,스틱, 무릅보호대, 물, 신발

---

**의류 / 신발**
- 트레일 등산화 (둘레길이라 본격 등산화보다 가벼운 것도 무방)
- 여벌 양말 (1일차 16.7km 장거리, 발 관리 중요)
- 우비 또는 방수 재킷 (지리산 특성상 갑작스러운 비 대비)
- 모자 (햇볕 차단)
- 봄/가을 기준 얇은 겹쳐입기용 미드레이어

**장비**
- 등산 스틱 (산 4개 종주, 내리막 무릎 보호)
- 배낭 (20~30L, 1박이므로 여벌옷 포함)
- 보조배터리 + 충전 케이블 (장시간 지도 앱 사용)
- 헤드랜턴 (1일차 7시간 코스, 늦을 경우 대비)

**행동식 / 수분**
- 물 1.5L 이상 (구간 내 보충 포인트 사전 확인)
- 행동식 (에너지바, 견과류, 김밥 등)
- 점심 도시락 또는 식당 위치 사전 파악

**기타**
- 개인 상비약 (진통제, 파스, 반창고, 물집 패치)
- 선크림
- 지퍼백 (우중 스마트폰·지갑 방수)
- 현금 (산간 구간 카드 불가 가능성)

### 고려사항

**이동 / 차량**
- 1일차 시작점(삼화실)과 2일차 종료점(원춘부)이 다름 → **차량 2대 필요하거나 택시/콜밴 사전 예약 필수**
- 숙소(켄싱턴 리조트 하동)에서 대축까지 차로 25분 → 아침 출발 시간 여유 확보

**일정 / 체력**
- 1일차 16.7km는 실제 5시간 예상이나 산 4개 포함 → 오전 일찍 출발 권장 (08:00 이전)
- 2일차는 8.6km / 2.5시간으로 여유 → 체크아웃 후 바로 출발 가능
- 전날 충분한 수면 및 숙소 조기 체크인 고려

**날씨**
- 지리산 둘레길은 숲 구간이 많아 우천 시에도 진행 가능하나 진흙길 미끄럼 주의
- 출발 전날 일기 예보 확인, 낙뢰 예보 시 일정 재검토

**숙소**
- 비바체 리조트 사전 예약 필수 (주말 성수기 조기 마감)
- 트레킹 후 짐 보관 또는 조기 체크인 가능 여부 리조트 확인
